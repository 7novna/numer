🔍 Таблица 5: Личный Разбор

📌 Формулы расчёта (согласно Excel):

Число Ума/Сознания: День рождения → свести к цифре
Пример: 15 → 1+5 = 6
Число Месяца: Месяц → свести к цифре
Пример: 03 → 0+3 = 3
Число Года: Год → свести к цифре
Пример: 1973 → 1+9+7+3=20 → 2+0 = 2
Число Действия/Миссии: Ум + Месяц + Год
Пример: 6+3+2 = 11 → 1+1 = 2
Число Реализации: Ум + Действие
Пример: 6+2 = 8
Итоговое число: Ум + Действие + Реализация
Пример: 6+2+8 = 16 → 1+6 = 7
🔑 КОД ЛИЧНОСТИ (4 цифры): Ум — Действие — Реализация — Итог
Пример: 6287
